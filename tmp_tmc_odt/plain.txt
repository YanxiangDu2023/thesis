Calculations
Step
Description
Preparation
There are 2 values for this step - P00 and P10. Preparation step has 2 steps - CRP data copy and processing and OTH data (Other data from files) copy and processing.CRP data has to be processed before OTH data(!). OTH data processing is not possible without CRP data. On this step we also prepare data for split â€“ copy into M00 calculation step data (with artificial machine lines CEX, GEC, GEW and WLO). For data from DES files, we are calculating FIDs based on MS% from file and value of total market from CRP.
Adjustment
This step creates calculations using calculation step A10. Its purpose â€“ to adjust values to have sum of all FIDs = TM. Here we do regard TMD source data, replace VCE data with one from SAL source; add adjustment value to make same with total market value from TMA. Formula: Adjustment value = Total Market â€“ Total Market Non VCE â€“ VCE; Assign Calculation Step as â€śA10â€ť; Assign Brand Code as â€śADJâ€ť.
Split
This step is M00. Here we do split artificial machine lines CEX, GEC, GEW and WLO into real TMC machine lines using TMA data as reference and certain rules. On this step we also check matrix Y44_TMC_TSICL [TMC] Size Class List by Brand (To be deleted) to delete non real combinations derived on this step.
Calculate Total Market (TM)
For certain machine lines, there is no total market value in TMA system. So in Calculate Total Market step, gather all data and delete double brand, then replace VCE FID from SAL, after all sum up all value to make final total market value. If machine line has source matrix information, then only data from primary and secondary source is considered as FID to be summed up.
Restatement
In Restatement step we gather all Volvo portfolio machine line (Only for restatement) data and delete double brand, then replace VCE FID from SAL, after all FIDs from OTH is recalculated based on ratio, to make final total market value. Final FIDs are recalculated using TM value â€“> sum of all FIDs = TM. This step is running calculation of new FIDs for country groups which are not donors or rest of (master data check).
Donor/ Rest of Hub
There is no market share information for some countries. For this case, HQ chooses which combination of geographical grouping can be base value for those countries, and TMC system creates FID data based on that information. Processing steps: deletion of double brand and then restatement. Additionally, CNX could be used if CNX is added as secondary source in a source matrix. All FIDs from CNX source must be confirmed values, so not for restatement. This step runs for country groups which have value R or D in their master data only.
Final Reports
Data must be uploaded into Reporting cube YD4CC045R to be available in Final reports. Transformation contains routine which derives Region and Market area from Country group matrix and process historical data to derive actual country group and region.
Data for TMC application is uploaded from CRP system (TMA, TMD, SAL) and from files (OTH).
Source files from other system have different formats, so HQ has to convert them with proper format to upload to TMC. Uploading function deletes all previous data by selected condition. After uploading, HQ can check results in â€śControl Report â€“ Raw Dataâ€ť menu. In this step, maintenance team had better check total FID value in files is same as one in control report.Every country has only one unique data source. We do upload by detailed data source(like CNX, DES) but in source matrix we use OTH for it.
